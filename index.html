API JavaScript (coxa-ui.js)

Inclua coxa-ui.js antes do fechamento do </body>. Todas as funções são globais.

FunçãoDescrição
toggleSidebar()Abre/fecha sidebar mobile
closeSidebar()Fecha sidebar mobile
toggleSec(btn)Expande/colapsa seção do sidebar
toggleDark()Alterna modo escuro + salva em localStorage
initDark(storageKey?)Inicializa dark mode a partir do localStorage
syncDarkIcon()Atualiza ícone do botão de dark mode
coxaToast(icon, title)Toast via SweetAlert2 (icon: 'success'|'error'|'warning'|'info')
coxaConfirm(form, title, text, icon, confirmText, color)Modal de confirmação com submit do form ao confirmar
openModal(id)Abre modal pelo ID
closeModal(id)Fecha modal pelo ID
switchLinksTab(btn, panelId)Troca aba no componente links-tabs
switchTab(btn, panelId)Troca aba no componente tab (sublinhado)
initTableSearch(inputId, tbodySelector?)Filtro ao vivo para tabelas
updateClock()Atualiza elemento #live-clock com hora atual (HH:MM:SS)
updateElapsed()Atualiza todos os [data-entrada="YYYY-MM-DD HH:MM:SS"] com tempo decorrido
mkSlug(text)Gera slug URL a partir de um texto
copyToClipboard(text, feedbackEl?)Copia texto para área de transferência
toggleSelRow(chk)Marca/desmarca linha na seleção em lote
toggleSelAll(chk)Marca/desmarca todas as linhas
updateSelBar()Atualiza contador e exibição da barra de seleção
clearSel()Limpa todas as seleções
Uso básico em Flask/Jinja2
